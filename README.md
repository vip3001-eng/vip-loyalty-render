# VIP غسيل وتلميع — موقع موبايل فقط (نقاط ولاء + قاعدة بيانات)

هذا مشروع جاهز (MVP) يعمل محليًا ويُرفع على أي سيرفر Node.js.
- واجهة عملاء (عميل وفي / عميل جديد)
- نافذة الشروط والأحكام (يتم تعديلها من الأدمن)
- توليد QR لكل زيارة (Pending) ثم اعتمادها من المحاسب/الأدمن
- نقاط (إضافة + استبدال)
- لوحة أدمن: إعدادات + مستخدمين + إشعارات + تصدير (Excel/Word)
- لوحة محاسب: اعتماد + استبدال
- قاعدة بيانات SQLite داخل المشروع

## التشغيل
1) ثبّت Node.js (LTS)
2) داخل المجلد:
```bash
npm install
npm run seed   # ينشئ مستخدم أدمن ومحاسب وإعدادات افتراضية
npm run dev
```

ثم افتح:
- العملاء: http://localhost:3000
- تسجيل دخول الإدارة/المحاسب: اضغط على شعار المغسلة 5 مرات بسرعة أو افتح http://localhost:3000/admin-login.html
